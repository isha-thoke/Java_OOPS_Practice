# Java OOPs Practice Project
This repository is dedicated to my Java learning journey and contains practical implementations of Object-Oriented Programming (OOPs) concepts.


🚀 Concepts Covered

In this project, I have implemented the following core OOPs principles:

* Classes & Objects: Defining the basic structure and blueprints of Java programs.
* Inheritance: Achieving code reusability using the extends keyword.
* Abstraction: Implementing high-level design using Abstract classes and Interfaces.
* Encapsulation: Ensuring data privacy and security through private variables and accessors (getters/setters).
* Polymorphism: Utilizing Method Overloading and Method Overriding for dynamic behavior.
* 
## 📂 Project Structure
Each file in this repository focuses on a specific concept or real-world logic:

1. Student.java: Handles student management logic and demonstrates the use of constructors and static methods.
2. Bank.java: A simple banking system implementation showcasing Encapsulation with private data fields.
3. Abstraction.java: Practical examples of how to use abstract classes.
4. Interface.java: Demonstrates how to define and implement multiple interfaces.
